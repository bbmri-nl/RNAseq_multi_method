# RNAseq multi method

## Requirements
- A conda installation
- A snakemake installation
- pandas (python3 library)
- cerberus (python3 library)
- biopet tools:
  - bamstats

## Included Mappers
- STAR
- STAR (2 pass mode)
- HISAT2 (doesn't work)

## Included Counting methods
- HTSeq-count
- featureCounts
- STAR QuantMode
